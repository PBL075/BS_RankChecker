BS_RankChecker
O aplicație interactivă în C# care determină rangul unui jucător de Brawl Stars în funcție de numărul de trofee introduse.

Ce face programul:
Întreabă utilizatorul: „Cate trofee ai?”.

Citește numărul de trofee de la tastatură.

Folosește structuri de control (if-else sau switch) pentru a încadra jucătorul într-o categorie de nivel (ex: Începător, Pro, Expert).

Afișează un mesaj personalizat bazat pe performanța utilizatorului.

Concepte C# aplicate:
Interacțiune cu consola: Utilizarea Console.Write și Console.ReadLine.

Conversia datelor: Transformarea textului citit în număr întreg Convert.ToInt32.

Logica decizională: Compararea valorilor pentru a stabili rangul corect.

Cum se rulează:
Descarcă fișierul Program.cs.

Rulează comanda dotnet run.
